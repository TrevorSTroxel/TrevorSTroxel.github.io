This will ba a tutorial on Arduino. We will start with where you download it, and end when you make a full loop on a circle Neo Pixel.
 1. The first thing that you will need to do is install arduino itself. Go to https://www.arduino.cc/en/main/software and download the apporite version for your computer.
 2. Once downloaded, open up the program and we will start with basic commands.
    a. To understand some of these commands, here some basic understandings of what they mean
        I. void setup(): This is the most basic part of any Arduino code. This allows the code to run in the first place.
        II. void loop(): This is the same as void setup(), but it allaows the code to repeat indefinauntly.
        III. int: Found in most codes in general. Specifies what ever you want to be a integer; ex. int clearPix, int drawmatix, int readButtons.
        IV. #define: This is used to set up the PIN number on the Neo Pixel and to tell how many pixels there are with NUMPIXELS.
        V. Serial.begin: This just allow any serial commands to run in the first place
        VI. Serial.print(ln): This is used to print, as in run the code, out the commands that you have set in place. There will sometimes be a ln at the end of Serial.print, this just means it prints it in a list.
        VII. for(): This makes any int values do a set command such as being a differnt color or being brighter than all others. ex. for(j = 0; j < 5 ; j++)
 3. If this seems confusing at first, don't worry. It will all start to make sence as we continue on with the code.
 4. We will now start off with doing the first part of making the circle.


 To start off, I will give you the begining part of the code because this should be the set up for most people:
 #include Adafruit_NeoPixel.h (A less than sign and a greater sign will be at the start and end of this respectivly, but this does work while writing this code)
 #ifdef __AVR__
 #include avr/power.h (A less than sign and a greater sign will be at the start and end of this respectivly, but this does work while writing this code)
 #endif
 #define PIN 6
 #define NUM_LEDS 16
 #define BRIGHTNESS 50
 Adafruit_NeoPixel strip = Adafruit_NeoPixel(NUM_LEDS, PIN, NEO_GRBW + NEO_KHZ800);
 void setup() {
 Serial.begin(115200); (This is for Trinket 5V 16MHz, you can remove these three lines if you are not using a Trinket)
 #if defined (__AVR_ATtiny85__)
 if (F_CPU == 16000000) clock_prescale_set(clock_div_1);
 #endif (End of trinket special code)
 strip.setBrightness(BRIGHTNESS);
 strip.begin();
 strip.show(); // Initialize all pixels to 'off'
 }
